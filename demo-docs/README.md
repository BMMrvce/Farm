# Demo Documents Pack

This folder contains client-facing printable documents in HTML format.

## Files

0. `index.html`
   - Branded single-screen click-through hub for previewing all reports and invoices.

1. `10-executive-requirement-report-print.html`
   - Executive requirement report in printable HTML.
2. `11-daily-operations-report-demo-print.html`
   - Auto-filled daily operations report in printable HTML.
3. `12-weekly-consolidated-report-demo-print.html`
   - Weekly consolidated report in printable HTML, including WhatsApp-to-report mapping.
4. `08-invoice-demo-software-service.html`
   - Auto-filled software invoice sample for live presentation.
5. `09-invoice-demo-farm-dispatch.html`
   - Auto-filled dispatch invoice sample for live presentation.
6. `assets/farmops-logo.svg`
   - Reusable logo file used in invoice templates.
7. `archive/`
   - Hidden master/template and markdown copies kept for backup and reuse, not shown in demo hub.

## Usage

- Open HTML files in a browser and print/export as PDF.
- Start with `index.html` for a one-screen walkthrough during demos.
- Markdown and master templates are archived under `archive/` and excluded from showcase navigation.
